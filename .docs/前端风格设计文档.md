# 应用设计风格规范指南

## 核心设计理念
1.  **流畅动效驱动交互**：通过平滑的过渡效果引导用户操作流程，让交互有“呼吸感”。
2.  **层级分明的内容组织**：采用“主容器-功能盒子-内容面板”三层结构，视觉层次清晰。
3.  **状态导向的视觉反馈**：每个交互元素都有明确的状态（默认/激活/悬停），使用户能直观感知系统响应。

## 布局规范
### 容器与盒子
- 主容器通常使用 `flex` 居中布局，占据整个视口高度。
- 内容盒子有固定尺寸（PC端）或自适应（移动端），包含圆角和投影。
- 内部模块采用绝对定位实现层叠或滑入效果，通过 `visibility` 控制显示。

### 响应式断点
```css
/* 大屏设备 (PC端) */
@media screen and (min-width: 651px) { /* 桌面端样式 */ }
/* 中屏设备 (平板) */
@media screen and (max-width: 650px) { /* 平板适配 */ }
/* 小屏设备 (手机) */
@media screen and (max-width: 400px) { /* 手机端微调 */ }
```
## 视觉设计体系
### 色彩方案
- **主色调**：`#7494ec` (晴空蓝) - 用于重点背景和品牌元素
- **辅助色**：`#ffffff` (纯白) - 内容区域背景
- **中性色**：`#e2e2e2` → `#c9d6ff` (渐变灰蓝) - 页面背景渐变
- **文本色**：`#333333` (深灰) - 主要文字，`#ffffff` - 浅色背景文字
- **交互色**：悬停时使用半透明叠加 `rgba(255, 255, 255, 0.2)`

### 圆角与投影
- **大圆角**：`30px` (容器级) - 营造柔和友好的氛围
- **中圆角**：`8px` (按钮级) - 保持可点击性的同时不失圆润
- **小圆角**：`20vw` (移动端背景) - 响应式圆角
- **投影**：`box-shadow: 0 0 30px rgba(0, 0, 0, 0.2)` - 柔和弥散投影

### 间距系统
- **大间距**：`40px` (PC端内边距) - 保证内容呼吸空间
- **中间距**：`20px` (移动端内边距/段落间距)
- **按钮尺寸**：`160px × 46px` (PC端标准按钮)

## 动效设计规范
### 过渡效果
```css
/* 慢速平滑过渡 (用于主要状态切换) */
transition: 1.8s ease-in-out;
/* 中速过渡 (用于面板切换) */
transition: 0.6s ease-in-out;
/* 延迟策略 */
transition-delay: 0.6s; /* 第一级延迟 */
transition-delay: 1.2s; /* 第二级延迟，创造交错效果 */
```

### 动效使用场景
1.  **布局转换**：使用长持续时间(1.8s)实现流畅的模式切换
2.  **元素显现**：通过 `visibility` + 位置动画实现淡入淡出
3.  **层次移动**：通过延迟创造“先来后到”的视觉层次
